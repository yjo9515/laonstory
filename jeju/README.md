# How to start

1. 프로젝트 터미널에 `dart run build_runner build --delete-conflicting-outputs` 로 generated 파일 생성 (style.g.style 에러 invaildtype에러 발생시 invalildtype 전체를 Color로 변경해야됩니다)
2. 타겟을 앱으로 실행
